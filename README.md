# Diabetes Clinical Outcome Analysis
## Overview
This project analyzes the Pima Indians Diabetes Dataset to investigate clinical factors associated with diabetes outcomes. The analysis focuses on understanding how variables such as glucose levels, BMI, age, and insulin relate to diabetes status.
## Objectives
- Clean and explore the clinical dataset
- Identify patterns in key metabolic indicators
- Examine associations between patient characteristics and diabetes outcome
- Interpret findings from a clinical perspective
## Dataset
**Source:** Pima Indians Diabetes Dataset (UCI Machine Learning Repository)  
**Sample size:** 768 patients  
**Variables include:**
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Diabetes status)
## Clinical Questions Explored
1. Which BMI ranges show the highest prevalence of diabetes?
2. How strongly are glucose levels associated with diabetes outcomes?
3. Are age or metabolic indicators stronger predictors of diabetes status?
4. How do insulin levels differ between diabetic and non-diabetic patients?
5. How does blood pressure distribution differ between diabetic and non-diabetic patients?
6. Does the Diabetes Pedigree Function (family history score) show a meaningful pattern with outcome?
## Methods
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualization of clinical variables
- Statistical analysis of risk factors
